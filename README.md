# 📚 My Library App

A simple, elegant book library web app built with HTML, CSS, and vanilla JavaScript. Users can add books, mark them as read/unread, and remove them from their personal library.

---

## ✨ Features

- Add new books with title, author, pages, and read status
- Toggle books between "Read" and "Unread"
- Remove books from the library
- Dynamic UI updates (no page reloads)
- Responsive grid layout
- Clean, minimal UI with a vintage library aesthetic
- Form validation for required fields

---

## 🧠 How It Works

Books are stored in a JavaScript array and rendered dynamically into the DOM. Each book is an object created using a `Book` constructor, with unique IDs for easy manipulation.

---

## 📂 Project Structure

index.html → Main structure
style.css → UI styling and layout
script.js → App logic and interactivity

---

## 🚀 Getting Started

1. Clone or download the project
2. Open `index.html` in your browser
3. Click **Add Book** to start building your library

---
